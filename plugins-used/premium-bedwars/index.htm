Code (Text):
1. Set the lobby location with command: /bedwars setLobby
2. Create an arena with command: /bedwars create <arenaName> <arenaType> <minPlayers>
3. Place a sign :
Bedwars
arenaName
4. Enjoy the plugin

Scoreboard placeholders:

Waiting:

  • <mapName> – displays the map name
  • <max> – displays the number of max players
  • <inGame> – displays the number of players in game
  • <time> – displays time to start
  • <min> – displays the number of min players

Game:

  • <mapName> – displays the map name
  • <inGame> – displays the number of players in game
  • <max> – displays the number of max players
  • <min> – displays the number of min players
  • <isAliveTeamName> – displays if the team is alive
  • <nextEvent> – displays the next event
  • <time> – displays the time of next event
  • <day> – displays the day of the month
  • <month> – displays the month of the year
  • <year> – displays the year
  • <finalKills> – displays your final kills
  • <kills> – displays your kills
  • <deaths> – displays your deaths
  • <bedsBroken> – displays your beds broken

Holograms placeholders:

  • <totalBedsBroken> – displays total beds broken
  • <totalBedsLost> – displays total beds lost
  • <totalKills> – displays total kills
  • <totalFinalKills> – displays total kills
  • <totalDeaths> – displays total deaths
  • <totalGamesPlayed> – displays total games played
  • <totalWins> – displays total wins
  • <totalLosses> – displays total losses
