 Our website is now mobile friendly, when
 a mobile device or tablet accesses the site,
 an alternative site will be sent where
 certain elements have been scaled down
 specifically for mobile devices, though we
 still recommend using a PC for the best
 experience, for example posts on the
 site when accessed on a PC are elements
 with a layout and text, when as the
 mobile version the posts are replaced with
 scaled down images https://floydcraft.co.uk/

 This works by making use of the 301
 redirect, if a mobile device is detected
 even requesting a desktop site it will
 be automatically redirected to the mobile
 friendly page.
