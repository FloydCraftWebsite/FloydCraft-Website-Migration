 Bedrock is now natively supported in
 FloydCraft, no 3rd party app needed, simply
 open for example Minecraft Windows 10
 Edition and instead of connecting on the
 normal IP instead join floydcraft.online
 that will take you to the same server but
 on bedrock where you can play side by side
 with java edition players, you will be
 prompted to login with your java edition
 minecraft account when joining, this is to
 let you play as you even if your bedrock
 username is different.

 If you need Offline mode or access on a
 console e.g. Xbox One, Playstation 4 or
 Nintendo Switch, you will still need to
 use our Bedrock Lan Proxy app, ask
 on Discord if you need an update.
