 For Admins:
 As an admin you can now create your own Spyro
 style wirlwinds in areas of the server with the command
 /gt create <height> <power>
 Height is how many blocks up the wirlwind will go.
 Power is how fast the player will travel.
 The wirlwinds will be set in the area you are looking at
