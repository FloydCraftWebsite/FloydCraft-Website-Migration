Basic Information
FloydCraft has been around since way back in 2011! we started on the domain floydcraft.servegame.com from early 2011 to late 2014, we then bought floydcraft.co.uk and have been using that ever since, We have a small community of regular players that are friendly and always looking for new Faction members.
Please note that as we run our server in Offline Mode to allow less fortunate players to join, we have an Authentication system in place,  it’s very simple to use, once you join you will have 2 minutes to /register <password> <password> after doing that you will be successfully logged in, the next time you join you will then need to type /login <password>
If you forget your password you can contact us on Discord or the Forum and request a Password Reset.
Because we have been running this for so long we’ve gained some great knowledge and have been able to produce our own plugins some of which can be downloaded from Spigot
e.g. WeatherNews. You can also find detailed information on every plugin we have used in the past and now on our website here  ___________________________________________________________________________________________

What do i do on FloydCraft
When you first join you will appear in the servers main Spawn location, the Spawn is a large building that has portals, an arcade and all the shops. first of all i would Recommend grabbing the kit from the Kit NPC in spawn

After claiming the kit (or not) you can start your adventure, You can play in Survival worlds or Creative Plots from the start, to see all the worlds you can travel to simply type /wc, that will open up the Warp Clock GUI, of course that is entirely Optional, you can simply run right out of spawn and into the EpicWorld.
if you decided to switch to Creative Plots when you first joined you will notice that the plots are HUGE! The PlotWorld has its own builder ranks that are separate to the ranks of the survival worlds, You will also have WorldEdit from the start. If you stayed in survival then you will have a lot of fun to come if you give it a chance to get used to everything, There is way over 1000 commands that will keep you busy for hours on end.. if you decide to learn them all that is..

You can Teleport to a random location in all survival worlds by typing /wild or /rtp, once you find a nice spot you like you can set a home with the command /sethome, to return there type /home
You can set upto 3 homes if you give them all separate names e.g. /sethome base then /home base. It is likely at this point you will have already been invited to a Faction by one of the Faction Owner Players, If not though you can create your own with the command /f create <name>
you will then be able to claim your land with the command /f claim one. for more faction information click here. now you’ve done that you will want to start earning some money  to buy items from the shops and spend on heads, to get money the easiest way would be to get a job, to view a list of jobs type /jobs list and then to join the job you like the look of type /jobs join <NameOfJob> you will then receive money based on the job every other InGame day.
There’s a HUGE amount more things you can do in FloydCraft that isn’t listed here because of how large this list would be, You can explore our website for more information on this topic if needed.
