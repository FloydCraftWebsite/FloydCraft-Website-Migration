✓ BetterMcmmoStats [McMMOStatsGui] 1.5

No permission to download
  • Author sdfsdfsd
  • Creation date Aug 7, 2017
Overview History Discussion
BetterMcmmoStats (earlier known as McMMOStatsGui) is a simple but effective plugin that lets you choose between 2 unique ways of showing your mcmmo stats. A GUI mode and a CHAT mode. See pictures below.

Features:
- Choose between a nice looking GUI or text in chat to show your mcmmo levels.
- Chat mode is very customizable.
- Change both names and lores of items in the GUI in the config.
- Change GUI Title
- In-game config reload with /stats reload for easy customization
- Supports Java 6, 7 and 8.

Pictures:


Permissions:
There's none, it's all ready to go by default!

Commands:
/stats <name>
/mcstats <name>
